# Лабораторна робота №2: Наука про дані: підготовчий етап

Ця робота присвячена базовому аналізу, очищенню, нормуванню та первинній обробці великих масивів даних за допомогою мови програмування Python та бібліотеки Pandas.

## Структура проєкту
* `vhi_analysis.ipynb` — Модуль 1: Аналіз індексів вегетації (VCI, TCI, VHI) за роками та областями, реалізація процедур вибірки та екстракції екстремальних значень посух.
* `electricity_consumption.ipynb` — Модуль 2: Обробка датасету енергоспоживання (вибірки за часом, підрахунок середніх значень, фільтрація за потужністю, обчислення коефіцієнтів кореляції Пірсона та Спірмена, One-Hot Encoding).

## Вимоги до системи
* **Мова:** Python 3.10+
* **Основні бібліотеки:** `pandas`, `numpy`, `urllib`, `scipy`
* **Середовище виконання:** Jupyter Notebook / JupyterLab
