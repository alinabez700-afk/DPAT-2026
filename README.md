
# Лабораторна робота №3: Візуалізація багатовимірних даних

Дослідження присвячене аналізу унікального багатовимірного датасету з репозиторію UCI, проведенню процесу очищення даних (Data Cleaning) та побудові комплексних графіків для виявлення прихованих закономірностей.

## Вимоги до операційної системи та заліза
* **ОС:** Будь-яка десктопна платформа (Windows, Linux, macOS)
* **Залежності:** Python 3.10+, `matplotlib`, `seaborn`, `pandas`, `numpy`, `jupyter`

## Функціонал та побудовані графіки (Всього: 6 графіків)
1. **Lineplot / Scatter plot:** Залежність між числовими атрибутами датасету.
2. **Гістограма розпреділення:** Відображення кількості елементів, розбитих строго на 5 користувацьких діапазонів.
3. **Багатовимірні графіки (Multi-dimensional plots):** Візуалізація зрізів даних із кодуванням додаткових вимірів за допомогою кольору (hue) та розміру точок.
4. **Матриця кореляції (Heatmap):** Візуальне відображення зв'язків між усіма числовими ознаками.
