# Лабораторна робота №5: Наука про дані: обмін результатами через Streamlit

Інтерактивний веб-додаток, побудований на фреймворку Streamlit, який виступає в ролі дашборду для аналізу та демонстрації результатів дослідження індексів вегетації (VCI, TCI, VHI).

## Структура веб-інтерфейсу
* **Панель фільтрів (Sidebar):**
  * Dropdown для вибору індексу (VCI/TCI/VHI).
  * Dropdown для вибору конкретної області дослідження.
  * Слайдери для вибору діапазону тижнів та років.
  * Чекбокси для сортування даних за зростанням або спаданням.
  * Кнопка автоматичного скидання всіх встановлених фільтрів.
* **Вкладки результатів (Tabs):**
  1. ` Таблиця даних` — Динамічний вивід відфільтрованої та відсортованої матриці.
  2. ` Графік часового ряду` — Візуалізація динаміки обраного індексу в часі.
  3. ` Порівняння областей` — Порівняльний аналіз середнього значення обраного індексу по всіх регіонах.
