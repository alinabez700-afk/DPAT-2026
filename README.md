# Лабораторна робота №4: Візуалізація даних та цифрова фільтрація сигналів

Програма з інтерактивним графічним інтерфейсом, яка дозволяє моделювати математичну функцію гармоніки виду $y(t) = A \cdot \sin(\omega t + \varphi)$ із можливістю динамічного накладання випадкового шуму та подальшою цифровою фільтрацією сигналу.

## Керування та інтерфейс
* **Слайдери:** Зміна амплітуди ($A$), частоти ($\omega$), фазового зсуву ($\varphi$), а також математичного очікування та дисперсії шуму.
* **Чекбокси:** Перемикання відображення чистого/зашумленого сигналу та активація фільтра.
* **Кнопка «Reset»:** Миттєве відновлення початкових параметрів системи.

## Алгоритм фільтрації
Для очищення зашумленого сигналу інтегровано цифровий фільтр із бібліотеки `scipy.signal` (фільтр IIR / Баттерворта). Після фільтрації отримана гармоніка автоматично відображається на графіку поверх зашумлених даних для порівняння результатів.
