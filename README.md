# Лабораторна робота №6: C++ та OpenCV

## Вимоги до ОС та заліза
* **ОС:** Linux (Kali Linux / Ubuntu)
* **Компілятор:** GCC/G++ (з підтримкою C++17), CMake
* **Залізо:** Веб-камера доступна за шляхом `/dev/video0`

## Інструкція для збірки та запуску
Програма автоматично збирається за допомогою трьох послідовних команд:
1. `./preinstall.sh`
2. `./build.sh`
3. `./run.sh`

## Керування
* `0` — Звичайний режим
* `1` — Інверсія кольорів
* `2` — Розмиття Гауса
* `3` — Фільтр Кенні
* `4` — Глітч ефект
* `Слайдер` — Зміна яскравості
* `ESC` — Вихід
